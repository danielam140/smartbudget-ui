# SmartBudget – Interfaz Web

## Descripción del proyecto

SmartBudget es una maqueta funcional de una aplicación web orientada a la visualización y gestión de finanzas personales.
Este proyecto corresponde al **Módulo 3: Desarrollo de la Interfaz de Usuario Web**, y tiene como objetivo implementar una interfaz navegable, responsiva y escalable, alineada a un diseño previamente definido, aplicando buenas prácticas de desarrollo front-end.

El alcance del proyecto se centra en la **implementación de la interfaz**, considerando estructura semántica, organización de estilos, layout responsivo e integración de frameworks CSS.

---

## Objetivo

Construir una interfaz web funcional y modular que represente la landing page del sistema SmartBudget, con potencial de escalar hacia una aplicación completa, utilizando herramientas modernas de desarrollo front-end.

---

## Tecnologías utilizadas

* **HTML5** (estructura semántica)
* **CSS3**
* **SASS / SCSS** (arquitectura 7–1)
* **Bootstrap 4** (componentes y sistema de grillas)
* **Flexbox y Grid** para layout responsivo

---

## Metodología de estilos

Se utilizó la metodología **BEM (Block, Element, Modifier)** para la organización de clases CSS, lo que permite:

* Mejor legibilidad del código
* Escalabilidad del proyecto
* Separación clara de componentes

Esta metodología se complementa con el uso de SASS para modularizar los estilos y evitar duplicaciones.
